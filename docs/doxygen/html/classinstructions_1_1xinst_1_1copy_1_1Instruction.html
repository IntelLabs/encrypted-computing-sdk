list instructions.xinst.copy.Instruction.parseFromPISALine (   cls,
str  line 
)

Parses a copy instruction from a P-ISA Kernel instruction string.

Parameters: line (str): String containing the instruction to parse. Instruction format: N, copy, dst (bank), src (bank), res=0 # comment Comment is optional.

Example line: "13, copy, output_0_1_3 (2), c_0_1_3 (0), 0"

Returns: Namespace: A namespace with the following attributes: N (int): Ring size = Log_2(PMD) op_name (str): Operation name ("copy") dst (list[(str, int)]): List of destinations of the form (variable_name, suggested_bank). This list has a single element for copy. src (list[(str, int)]): List of sources of the form (variable_name, suggested_bank). This list has two elements for copy. res: Residual for the operation. Ignored for copy/move comment (str): String with the comment attached to the line (empty string if no comment).
