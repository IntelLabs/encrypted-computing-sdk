Detailed Description

Represents a CInstruction, which is a type of BaseInstruction.

This class provides the basic structure and functionality for CInstructions, including methods for converting to CInst ASM-ISA format.

Attributes: id (int): User-defined ID for the instruction. throughput (int): The throughput of the instruction. latency (int): The latency of the instruction. comment (str): An optional comment for the instruction.
