int linker.MemoryModel.useVariable (   self,
str  var_name,
int  kernel 
)

Uses a variable, decrementing its usage count.

If a variable usage count reaches zero, it will be deallocated from HBM, if needed, when a future kernel requires HBM space.

Parameters: var_name (str): The name of the variable to use. kernel (int): The kernel that is using the variable.

Returns: int: The HBM address for the variable.
