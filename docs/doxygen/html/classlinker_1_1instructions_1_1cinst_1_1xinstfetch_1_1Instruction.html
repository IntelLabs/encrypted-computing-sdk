Detailed Description

Encapsulates a xinstfetch CInstruction.

This instruction fetches instructions from the HBM and sends them to the XINST queue.

For more information, check the xinstfetch Specification: https://github.com/IntelLabs/hec-assembler-tools/blob/master/docsrc/inst_spec/cinst/cinst_xinstfetch.md

Properties: dstXQueue: Gets or sets the destination in the XINST queue. srcHBM: Gets or sets the source in the HBM.
