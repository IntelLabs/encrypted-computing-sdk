Detailed Description

Encapsulates kg_load CInstruction.

kg_load instruction loads HW-generated key material from the keygen engine into a CE data register.

To start the keygen engine, a seed should be loaded followed by a kg_start instruction to start the key material generation.

kg_load, dst_register

Rules:

  1. kg_loads and kg_starts must be latency cycles apart from any other kg_load and kg_start. It takes between 10 to latency cycles for the key generation resource to generate the next key material, possibly causing contention if the key material is requested by any kg_load within latency cycles.
