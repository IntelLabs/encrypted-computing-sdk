HERACLES Kernel Language extension for VS Code

This is an experimental first attempt at creating a VS Code extension that understands the grammar of the HERACLES Kernel Language.

<mark> This is experimental code. Use it at your own risk. </mark>

Installation

A pre-packaged version of this extension '.vsix' file is avialble in the current folder. You can manually install using the Install from VSIX command in the Extensions view command dropdown. For an alternative way to install a local VS Code extensions, please check the VS Code documentaiton.

Features

The only available feature is the toggle comments on a line
