197 class Register(CycleTracker):
198  """
199  Represents a register in the register file.
200 
201  Inherits from CycleTracker to manage the cycle when the register is ready to be used.
202  This class tracks the register name, the variable contained within the register as a form
203  of inverse look-up, and whether the register contents are "dirty".
204 
205  A register is identified by its bank and index inside the bank. The name of the
206  register is formatted as `r<register>b<bank>`. For example, register 5 in bank 1 has the name `r5b1`.
207 
208  Properties:
209  bank (RegisterBank): The bank where this register resides.
210  name (str): The name of this register, built from the bank and register indices.
211  register_index (int): The index for this register inside its bank.
212  register_dirty (bool): Specifies whether the register is "dirty". A register is dirty if it has
213  been written to but has not been saved into SPAD.
214  contained_variable (Variable): The variable contained in this register, or None if no variable is currently
215  contained in this register. This is used as a form of inverse look-up.
216  """
