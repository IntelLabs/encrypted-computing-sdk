Constructs a new AssemblerRunConfig Object from input parameters.

See base class constructor for more parameters.

Args: input_file (str): Input file containing the kernel code to assemble. Kernel code should have twiddle factors added already as appropriate. input_mem_file (str): Optional input memory file associated with the kernel. If missing, the memory file is expected to be same as input_file, but with extension ".mem". output_dir (str): Optional directory where to store all intermediate files and final output. This will be created if it doesn't exists. Defaults to the same directory as the input file. output_prefix (str): Optional prefix for the output file names. Defaults to the name of the input file without extension.

Raises: TypeError: A mandatory configuration value was missing. ValueError: At least, one of the arguments passed is invalid.
