Detailed Description

Represents a memory-level instruction (MInstruction).

This class is used to encapsulate the properties and behaviors of a memory-level instruction, including its throughput, latency, and a unique counter value that increases with each MInstruction created.

Methods: count: Returns the MInstruction counter value for this instruction.
