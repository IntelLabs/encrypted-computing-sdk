Detailed Description

Represents a cexit CInstruction.

This instruction terminates execution of a HERACLES program.

For more information, check the cexit Specification: https://github.com/IntelLabs/hec-assembler-tools/blob/master/docsrc/inst_spec/cinst/cinst_cexit.md
